Installation

No separate install step.
Clone the repository, open the folder, and run it with a static server.

Example:
Right click index.html
Choose “Open with Live Server” in VS Code.

Usage

The Smart Select is a reusable multi select component.

Examples:
Use it for facilities in lending.
Use it for accounts in payments.
Use it for portfolios in wealth.

Only the data and getId/getLabel mappings change.
The component, behavior, and events stay the same.

Testing

Start a static server from the project root.
Open tests/tests.html in the browser.
The page shows OK or FAIL for each test case.
